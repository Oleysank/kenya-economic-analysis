# Kenya Agricultural Production Analysis 
A study on food security resilience and sectoral divergence (1961-2021) 
## Key Data Insights 
* **Growth:** Maize production has seen a **3.5x growth multiplier** since 1961. 
* **Crisis Impact:** Identified 15 years with production drops , totaling **6.5M+ tonnes lost**. 
* **Worst Year:** 1984 experienced a catastrophic **-38.2% shock**.
